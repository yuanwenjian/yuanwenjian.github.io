Mybatis SqlSession 四大对象 Executor 详解

发表于 2019-08-04 | 分类于 Mybatis ， Java |
Mybatis通过SqlSession进行sql的执行,sql执行过程中涉及四种对象,及Executor,StatementHandler,ParameterHandler,ResultHandler.其中ParameterHandler处理sql参数,StatementHandler调用数据库statement执行,ResultHandler 对返回值封装,Executor mybatis执行器,统一对另三个对象就行调度

Mybatis通过SqlSession进行sql的执行,sql执行过程中涉及四种对象,及Executor,StatementHandler,ParameterHandler,ResultHandler.其中
ParameterHandler处理sql参数,StatementHandler调用数据库statement执行,ResultHandler 对返回值封装,Executor mybatis执行器,统一对另三个对象就行调度

Executor 配置

ExecutorType 有三种类型 分别为
SIMPLE, SimpleExecutor 简单执行器,每次执行都会新建Statement
REUSE,ReuseExecutor 重复利用之前创建好的Statement的对象,如果sql语句相同的话
BATCH,BatchExecutor 批量执行所有更新语句

总结

  1. mybatis 通过在节点配置defaultExecutorType,
  2. 在创建Configuration ,解析defaultExecutorType ,创建对应的ExecutorType作为configuration中ExecutorType 属性,默认为SIMPLE
  3. 创建SqlSession 过程是通过configuration 属性ExecutorType ,创建对应的Executor类型作为executor属性
  4. 在Sqlsession 执行sql实际调用属性executor对应方法
