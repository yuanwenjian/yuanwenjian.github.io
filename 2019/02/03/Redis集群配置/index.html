重新分片

使用以下命令分配

  1. 重新分配哈希槽

  2. 输入哈希槽个数 出现以下提示 表示你想要为新增的节点添加多少哈希槽 16384/节点个数 输入

  3. 为哪个节点分配, 查看节点信息，输入对应的hash值

  4. 选取分配的源节点，就是从哪个节点移动 ，平均分配的话输入all

  5. 接下来输入确认移动即可

  6. 完成后查看节点信息
